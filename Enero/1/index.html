Introduccion a JavaScript

¿Que es Javascript?

JavaScript es un lenaguaje de programación interpretado, dialecto del estándar ECMAScript. Se defiene como orientado a Objetos, basado en prototipos, imperativo, debilmente tipado y dinámico

Fuente

Javascript fue creado para dar vida a las paginas Web

Los programas en este lenguaje se llaman script. Se pueden escribir directamente en el HTML de una pagina web y ejecutarse automaticamente a medida que se carga la pagina

Los scripts se proporcionan y ejecutan como texto plano. no necesitan preparción especial o compilacion para correr
